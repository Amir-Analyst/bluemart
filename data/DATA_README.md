# Large Data Files

This directory contains large CSV files that are excluded from the Git repository due to GitHub's 25MB file size limit.

## Excluded Files

### Raw Data
- **bm_sales.csv** (~604 MB) - Raw sales transactions (11.2M records)
  - Generated by: `scripts/generate_data.py`
  - Contains: Daily sales transactions for 2025

### Processed Data
- **sales_dashboard_data.csv** (~500 MB) - Aggregated dashboard data
  - Generated by: `scripts/process_data.py`
  - Contains: Processed and aggregated sales data for dashboard

## How to Generate These Files

If you clone this repository, you can regenerate these files by running:

```bash
# 1. Generate raw data
python scripts/generate_data.py

# 2. Process data for dashboard
python scripts/process_data.py
```

**Note**: Data generation may take 10-20 minutes and requires at least 8GB RAM.

## Included Files

The following smaller data files ARE included in the repository:
- `bm_stores.csv` - Store master data (50 stores)
- `bm_skus.csv` - SKU master data (5,000 SKUs)
- `bm_customers.csv` - Customer master data (~5,000 customers)
- `bm_promotions.csv` - Promotion calendar
- `bm_inventory.csv` - Inventory snapshot (~75,000 records)
- `summary_metrics.json` - Pre-calculated summary metrics

These files allow you to understand the data structure without needing to generate the full dataset.
